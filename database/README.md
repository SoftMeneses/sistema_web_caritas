# Base de Datos

Esta carpeta contiene todos los recursos relacionados con la base de datos del Sistema Web Cáritas.

## Estructura

schema/

Contiene el esquema principal de la base de datos.

triggers/

Contiene los triggers implementados en MySQL.

procedures/

Contiene los procedimientos almacenados.

## Convención

Los scripts SQL se numeran de forma incremental.

Ejemplo:

001_schema_inicial.sql

002_actualizacion_usuarios.sql

003_mejora_inventario.sql
